# Impto portfolio

A responsive, dependency-free developer portfolio built for GitHub Pages.

## Publish for free

1. Create a new **public** GitHub repository named exactly `impto-dev.github.io`.
2. Upload everything in this folder to the root of that repository.
3. In the repository, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then click **Save**.
6. After a minute or two, visit `https://impto-dev.github.io/`.

## Edit the content

- Page wording and links: `index.html`
- Colours and layout: `styles.css`
- Menu, scroll effects, and footer year: `script.js`

No build step or paid service is required.
